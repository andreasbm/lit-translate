## 1. Define the translations

Create a `.json` file for each language you want to support. Heres an example of how `en.json` could look like.

```json
{
  "header": {
    "title": "Hello",
    "subtitle": "World"
  },
  "cta": {
    "awesome": "{{ animals }} are awesome!",
    "cats": "Cats"
  },
  "footer": {
    "html": "<b>Bold text</b>"
  }
}
```
